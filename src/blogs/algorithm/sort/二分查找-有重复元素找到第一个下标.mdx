## 题目
请实现有重复数字的升序数组的二分查找:  
给定一个 元素有序的（升序）长度为n的整型数组 nums 和一个目标值 target  ，写一个函数搜索 nums 中的第一个出现的target，如果目标值存在返回下标，否则返回 -1.

数据范围： `0 <= n <= 100000`  

**实例1**:
```text
输入：[1,2,4,4,5],4
返回值：2
说明：
从左到右，查找到第1个为4的，下标为2，返回2   
```
**实例2**:
```text
输入: [1,2,4,4,5],3
返回值: -1
```
**实例3**
```text
输入：[1,1,1,1,1],1
返回值：0
```


## 代码：
```javascript
const search = (nums, target) => {
    let left = 0;
    let right = nums.length - 1;
    let result = []

    while (left <= right) {
        let mid = parseInt(left + (right - left) / 2);
        if (nums[mid] === target) {
            result.push(mid)
            right = mid - 1; // 找到后，再向数组左边找。
        } else if (target > nums[mid]) {
            left = mid + 1;
        } else if (target < nums[mid]) {
            right = mid - 1;
        }
    }

    return result.length === 0 ? -1 : Math.min(...result)
}
```
