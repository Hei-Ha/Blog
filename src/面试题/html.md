# HTML 相关面试题
## 1、HTML中的语义化标签
语义化标签是一种写 HTML 的方式，实现方法就是在写代码的时候，遇到标题就写 h1 —— h6
主要内容用 main, 文章用 article ,导航用 nav ，等等，这样的语义化明确了 HTML 的书写规范，按照这样写出来的代码，有利于搜索引擎检索，并且适合别的开发者阅读，有利于团队维护。

## 2、HTML 中有那些新标签
比如文章相关的标签： article，main, section, 
多媒体相关的标签：svg, canvas

## 3、svg 和 canvas 的区别是什么？
Canvas 主要是用笔刷来绘制图形的。
SVG 主要是用标签来绘制不规则矢量图的。
他们的相同点：都是主要用来画2D 图形的，
但是 SVG 节点过多的时候，会导致渲染很慢，Canvas 性能方面会更好一点，但是写起来比较复杂。
